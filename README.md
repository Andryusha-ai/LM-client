LM-client

Клиент для работы с локальными и удалёнными LLM/VLM-моделями через OpenAI-совместимый API.

Проект разрабатывается как персональный AI-ассистент с модульной архитектурой. Основная цель — сделать клиент, который не зависит от конкретной модели и может работать как с локальными, так и с удалёнными серверами.

Что уже умеет
обмен сообщениями с OpenAI-совместимыми API;
потоковый вывод ответа;
поддержка мультимодальных моделей (изображения);
прикрепление файлов к сообщениям;
просмотр изображений внутри клиента;
открытие прикреплённых файлов;
drag & drop вложений.
В разработке

В ближайших версиях планируется:

долговременная память;
работа с локальной файловой системой;
голосовой ввод и синтез речи;
поиск в интернете;
встроенный Python-интерпретатор;
поддержка нескольких пользователей;
мультиплатформенность.
Используемые технологии
Python
PySide6
OpenAI Compatible API
Статус проекта

Проект находится в активной разработке. Некоторые части архитектуры ещё меняются, поэтому возможны несовместимые изменения между версиями.

Главная идея проекта — возможность заменить используемую языковую модель без изменения клиентского приложения.
